# Changelog - Header

Registro de cambios por versión del módulo Header.

## V1 (actual / backup)

- **Archivo:** `Header.v1.tsx`
- **Estado:** Backup de la versión en producción.
- **Características:**
  - Header principal fijo (oculto al hacer scroll)
  - Sticky header desktop (píldora centrada) y sticky mobile
  - Logo, íconos: carrito (con contador), usuario, menú
  - Panel lateral: navegación, Mi Cuenta (Dashboard, Pedidos, Wishlist, Direcciones, Logout), Login, idioma, tema
  - Props: `isScrolled`, `language`, `isDarkMode`, `isMobileMenuOpen`, `onNavigateAccount`, `onNavigateCart`, etc., `translations`
