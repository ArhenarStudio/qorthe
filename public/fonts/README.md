# Fuentes del sitio

El sitio usa la tipografía de marca **ITC Avant Garde Gothic Pro**:

| Archivo | Uso |
|---------|-----|
| `ITCAvantGardeGothicPro-Regular.woff2` | Peso normal (400) |
| `ITCAvantGardeGothicPro-Bold.woff2` | Peso bold (700) |

Los archivos deben tener exactamente estos nombres. Si faltan, el sitio usará Arial/Helvetica como fallback sin romperse.
